[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png)
Let us look at the architecture. When a user or account manager submits an account query, it first enters our security checkpoint node. This node scrubs sensitive PII like emails and phone numbers, blocks prompt injection attempts, and enforces discount authorization caps.
Next, the request hits root_agent, our central orchestrator. root_agent delegates telemetry analysis to churn_analyst_agent, which connects via MCP to fetch real-time usage metrics and support ticket escalations. If risk is detected, root_agent delegates to retention_strategist_agent to formulate a custom, policy-compliant discount and add-on package.
